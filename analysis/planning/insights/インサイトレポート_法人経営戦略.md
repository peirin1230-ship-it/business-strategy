# インサイトレポート — 法人経営環境を踏まえた大橋病院経営戦略
## 5視点統合分析フレームワーク

**作成日**: 令和8年4月11日
**分析テーマ**: 法人財務制約下での大橋病院の最適経営戦略
**前提資料**: 法人経営環境分析レポート、大橋病院経営戦略提言書

---

## インサイト1: 法人の「増収減益」構造と大橋病院の役割

### [データ] 数値事実

- 法人全体: 事業活動収入+32.4億円（+3.3%）に対し、支出+45.2億円（+4.6%）で△12.8億円の収支悪化（事業活動収支予算書）
- 医療経費: +15.6億円（+5.3%）の増加に対し、医療収入は+15.0億円（+2.0%）にとどまる（事業活動収支予算書）
- 大橋病院のみが粗利益改善: +122.4%（大森△32.9%、佐倉△26.4%）（月次分析レポート）
- 大橋の人件費: 前年比+0.3%（累計）と抑制に成功、収入は+3.6%成長（月次分析レポート）

### [臨床] 臨床現場の視点

大橋病院の収益効率が高い要因は、**臨床面での高回転・高密度モデル**にある。平均在院日数8.8日は同クラス平均12.9日の69%であり、入院期間中に高密度な診療を行い早期退院させるオペレーション力が背景にある。これは外科系を中心とした手術翌日からの早期リハビリ開始、クリニカルパスの整備、退院支援の早期介入によって実現されている。ただし、この高回転モデルは**新入院患者の安定供給**が前提であり、紹介件数の減少（特に整形外科△543件）は直接的にベッド回転の低下につながるリスクがある。

### [負荷] 現場負荷の評価

**負荷水準: ★★（中程度）**

高回転モデルの維持は、退院支援・入退院管理・クリニカルパス管理に関わるスタッフの継続的な業務負荷を伴う。ただし、大橋病院はこのモデルで既に数年間の運用実績があり、**日常業務として定着**している段階。新たな追加負荷というよりは、現行オペレーションの維持・微調整が主。

**緩和策**: 患者サポートセンターの強化（組織目標に明記）と院内救急搬送車を活用した転院支援の仕組みが既に機能しており、病床の流動性向上に寄与。

### [請求] 保険請求の視点

- **DPC期間II以内退院率の改善**: 短在院日数はDPCにおける係数改善に直結。機能評価係数IIの向上余地がある
- **入退院支援加算**: 早期退院支援と連動した算定（資料8でモニタリング済み）
- **急性期充実体制加算**: 全身麻酔3,261件/年で基準2,000件を大幅超過 → 維持は確実
- **地域医療支援病院**: 紹介率87.1%で要件（65%以上）を十分クリア
- **算定漏れリスク**: 高回転ゆえに入退院が多く、算定漏れの件数も相対的に多くなりやすい → 施設基準管理システムの導入が有効

### [経営] 経営アクション

**法人が「増収減益」の構造から脱却するためには、大橋モデルの3病院への波及が最も効果的。**

具体的には:
1. **大橋のベストプラクティス共有**: 高回転・高単価モデルのオペレーションノウハウを経営会議・戦略会議で共有
2. **DPCベンチマーク分析の深化**: 井上客員教授のDPC分析をさらに活用し、大橋の強みの要因分解と他院への適用可能性を検討
3. **収益インパクト**: 大橋の粗利益率を0.8%→3%に改善すると、約3.5億円/年の追加利益 → 法人全体の黒字転換に大きく貢献

## インサイト2: 大森再開発による投資配分の偏りと大橋の「自力更生」戦略

### [データ] 数値事実

- 3年間の設備投資: 306.8億円（法人全体収支計画）
- 大森再開発: 194.0億円（63.2%）（法人全体収支計画）
- 習志野新棟: 12.4億円（4.0%）
- 残り（通常設備投資）: 100.4億円/3年 = 年間約33億円（3病院+全学で分配）
- 大森2号館改修: 想定工費を大きく超過し計画見直し済み（上期事業運営状況）
- 3年間の手許資金取崩: 155.0億円（法人全体収支計画）

### [臨床] 臨床現場の視点

大型設備投資が不可能な中で、大橋病院が臨床面で成長するための切り口は**既存設備の高度活用と適応拡大**である。

- **ダヴィンチ**: 月平均10.2件で順調に稼働中。外科3術式+泌尿器科2術式の新規届出で適応拡大 → 手術件数・入院単価の上積み
- **ICU/HCU/SCU**: 6+12+6=24床の特定病床を最大活用 → 特定入院料の算定率向上が収益に直結
- **化学療法**: 外来化学療法加算の稼働は高水準 → 曜日別稼働の最適化で追加余地あり

新規の大型機器導入よりも、**既存機器の稼働率向上と新しい使い方の開発**が投資制約下での最適解である。

### [負荷] 現場負荷の評価

**負荷水準: ★★★（高い — ダヴィンチ拡大に関して）**

ダヴィンチ新5術式の導入は、手術室看護師と臨床工学技士に新たなトレーニング負荷を生む。ただし段階的導入が計画されており、一気に全術式を開始するわけではない。

**緩和策**:
- 術式ごとの導入スケジュールを明確化（四半期ごとに1-2術式の段階導入）
- 手術室の枠配分を見直し、ダヴィンチ用の定期枠を確保
- 既存のダヴィンチチームのスキルを活用した院内トレーニングプログラム
- タスクシフトの推進（臨床工学技士への機器操作支援の移管）

### [請求] 保険請求の視点

- **ダヴィンチ手術の保険適用**: 令和4年度改定で大幅に適用範囲が拡大。新規5術式が保険収載されていることを確認
- **手術支援加算**: ロボット支援手術加算（J038-3等）の算定要件の確認
- **急性期充実体制加算との連動**: 手術件数増は加算の維持・強化に寄与
- **査定リスク**: 新術式は審査機関の目が厳しい傾向。症例選定の適正化と手術記録の充実が重要

### [経営] 経営アクション

**「投資なき成長」のモデルを確立し、R10以降の投資余力回復に備える。**

1. **ダヴィンチ新5術式**: 年間追加36-60件 × 30-50万円 = 約1,080-3,000万円/年の増収（現場負荷を考慮し保守的に見積もり）
2. **特定入院料の算定率向上**: ICU/HCU/SCUの稼働率と算定率の最大化
3. **R10以降の計画**: 大森再開発完了後に大橋の設備更新計画を策定 → 3年間の収益改善実績をもとに法人内での投資配分を交渉する材料とする

## インサイト3: R8診療報酬改定 — 改定効果の「取りこぼし」を防ぐ

### [データ] 数値事実

- 急性期充実体制加算: 全身麻酔3,261件/年で基準2,000件を大幅超過（月次分析レポート）
- 地域医療支援病院: 紹介率87.1%（要件65%以上を十分クリア）（月次分析レポート）
- DPC期間II以内退院率: 改善傾向（月次分析レポート）
- 入退院支援加算: 算定状況をモニタリング済み（資料8）
- 外来化学療法加算: 高水準の稼働（資料2）

### [臨床] 臨床現場の視点

診療報酬改定への対応は、臨床面では主に以下の3つの軸がある:
- **施設基準の維持**: 急性期充実体制加算、地域医療支援病院、各種専門加算の要件充足
- **記録の適正化**: DPCコーディングの精度向上、重症度・医療看護必要度の正確な記録
- **新規加算の取得**: 改定で新設・要件緩和された加算の積極的な取得

大橋病院は「短在院日数・高入院単価」モデルであるため、DPC機能評価係数IIの効率性係数や複雑性係数において高い評価を得やすい構造にある。

### [負荷] 現場負荷の評価

**負荷水準: ★（低い）**

改定対応の主体は医事課であり、臨床現場への直接的な追加負荷は限定的。施設基準管理システムの導入（大森が先行導入済み）により、要件充足状況の自動モニタリングが可能に。

**緩和策**: 医事課スタッフへの改定内容研修を早期に実施し、算定漏れを防止。

### [請求] 保険請求の視点

- **施設基準の届出タイミング**: R8 Q1（令和8年4月）の早期届出が必須。届出の遅れは改定効果の取りこぼしに直結
- **DPC係数の最大化**: 機能評価係数IIの各項目（効率性、複雑性、カバー率等）を個別に検証し、改善余地のある係数に注力
- **算定漏れの典型例**:
  - 入退院支援加算の算定条件（退院時指導の記録不備等）
  - 栄養管理に関する加算（栄養サポートチーム加算等）
  - 外来化学療法加算の算定回数の最大化

### [経営] 経営アクション

**改定効果を「確実に」取り込むための3ステップ:**

1. **R8 Q1（4月）**: 施設基準の総点検と早期届出
2. **R8 Q2（7月）**: 算定実績の中間チェック。予定通りの増収が実現しているか検証
3. **R8 Q3-Q4（10月-3月）**: DPC係数の分析と次年度に向けた改善策の策定

## インサイト4: 外来患者数減少 — 「量の危機」と「質の転換」の分岐点

### [データ] 数値事実

- 1日平均外来患者数: 852人（予算910人、△58人、△6.4%）（月次分析レポート）
- 累計外来患者数: 前年比△8.5%の減少
- 外来単価: 22,841円（前年21,551円、+6.0%）
- 外来収入: 前年比+2.4%（単価上昇で量の減少を吸収）
- 初診率: 7.6%（目標10%、上期事業運営状況）
- 整形外科紹介: △543件（全科最大の減少）（月次分析レポート）
- 外来化学療法3科: +3,924万円/月の増収寄与（月次分析レポート）

### [臨床] 臨床現場の視点

外来患者数の減少には、**臨床的に望ましい側面と問題の側面の両方**がある。

**望ましい側面**:
- 逆紹介の推進による安定期患者の地域かかりつけ医への移行（地域医療支援病院の機能として適切）
- 外来単価の上昇は、より高度な医療が必要な患者に集中できていることの表れ

**問題の側面**:
- 整形外科の紹介激減（△543件）は、**入院手術の上流が枯渇する**リスクを孕む
- 初診率7.6%は新規患者の流入不足を示す → 将来の入院患者減につながる先行指標
- 外来単価の上昇率（+6.0%）は持続が難しい（薬価・材料費の高騰要因を含む）

### [負荷] 現場負荷の評価

**負荷水準: ★★（中程度 — 地域連携室に集中）**

外来患者数の回復には、紹介元医療機関への訪問活動の強化が中心。医師による訪問は月平均11件（目標達成済み）だが、整形外科に関しては追加的な訪問・セミナー開催が必要。

**緩和策**:
- Web予約システム（やくばと）の導入により、紹介受付の効率化と紹介元の利便性向上
- AI電話応答システム（大森で導入済み）の大橋への展開で電話対応負荷の軽減
- 広報委員会の活動強化（HP閲覧+5万件の成果を継続）

### [請求] 保険請求の視点

- **外来化学療法加算**: 高単価の主要因。曜日別稼働の最適化で算定回数を増やす余地
- **外来リハビリテーション**: 資料5でモニタリング済み。単位数の最大化を図る
- **紹介受診重点医療機関**: 紹介率87.1%は要件を十分クリア → 外来機能の分化を保険請求面でも活用
- **逆紹介患者の再紹介率**: 逆紹介後に症状が悪化して再紹介される率をモニタリングすることで、逆紹介の適正性を検証可能

### [経営] 経営アクション

**外来患者数の目標: R8末までに1日平均880人、R9末までに900人への回復**

施策の優先順位:
1. **整形外科の紹介回復**（インパクト最大: △543件の回復で月+2,492万円の増収）
   - 紹介元への集中訪問（月15件に増加）
   - 整形外科の強みを活かした医療連携セミナーの開催（四半期1回）
   - 紹介減少の原因分析（医師変動・競合分析・疾患構造変化）
2. **Web予約システムの効果最大化**（新規紹介のハードルを下げる）
3. **外来化学療法の拡大**（量減少を高単価で補う戦略の継続）
4. **広報強化**（HP閲覧+5万件の勢いを維持、ダヴィンチ等の先進医療を訴求）

## インサイト5: 法人の組織目標体系から見た大橋病院の戦略整合性

### [データ] 数値事実

- 法人の5つの基本方針: ①アイデンティティ・クオリティ、②ガバナンス、③財政基盤強化、④DX推進、⑤災害対策（上期事業運営状況）
- 法人目標（R8）: 「3病院の収支管理・経営分析に基づき経営課題に迅速に対応」「基本金組入前当年度収支差額の改善に注力」（組織目標 各所属）
- 大橋目標（R8）: 6項目の組織目標と10項目の具体策（組織目標 各所属）
- 経営会議: R7は3回開催予定。第1回は大橋6/24に開催済み（上期事業運営状況）
- 3病院合同戦略会議: R7年11月に開催検討中（上期事業運営状況）

### [臨床] 臨床現場の視点

法人が「高度急性期医療の提供を重点」とする方針は、大橋病院の臨床戦略と完全に合致している。特に:
- **ダヴィンチの適応拡大**: 法人の「ロボット支援手術をはじめとする高度医療の提供拡大」方針に直結
- **救急強化**: 区西南部の中核病院としての救急医療連携は法人の社会的責任の一端
- **タスクシフト/シェア**: 法人が「診療看護師、臨床工学技士、医師事務作業補助者の増員」を推進 → 大橋の働き方改革を後押し

### [負荷] 現場負荷の評価

**負荷水準: ★（低い — 戦略の整合性確認は事務レベルの作業）**

法人の組織目標と大橋の具体策の整合性は高く、新たな方向転換や大幅な追加施策は不要。既に走り出している施策の着実な実行が最重要。

### [請求] 保険請求の視点

法人レベルで推進される医療DX施策が、保険請求の効率化に直結する:
- **電子処方箋**: 導入完了 → 処方箋管理の効率化
- **施設基準管理システム**: 大森で先行導入 → 大橋への展開で算定漏れ防止
- **電子カルテ情報共有サービス**: 検討中 → 地域連携の深化と紹介・逆紹介の効率化
- **データ定義の統一**: R7より3病院間のデータ定義統一を実施 → 経営分析の精度向上

### [経営] 経営アクション

**法人の組織目標体系における大橋病院の「貢献の見せ方」が重要。**

1. **経営会議・戦略会議での発信**: 大橋の粗利益改善（+122.4%）の成功要因を定量的にプレゼンし、法人内での評価を確立
2. **「自力更生」の実績づくり**: 大森再開発期間中に、投資なしで収益改善を実現した実績は、R10以降の投資配分交渉で強力な材料になる
3. **法人施策の積極活用**: 3病院共同の医療経費削減、看護師確保、DX推進を最大限活用し、大橋単独では実現困難な施策を法人の力で推進
4. **R10以降への布石**: 大森再開発完了後の次期中計で大橋への投資配分を獲得するため、設備更新計画の策定を早期に開始（R9 Q3-Q4が適切なタイミング）
