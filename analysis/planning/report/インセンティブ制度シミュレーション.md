# 大橋病院 インセンティブ制度シミュレーション
## — 人事評価連動型 + KPI連動型の二層構造 —

**作成日**: 令和8年4月11日
**前提**: 恒久的なベースアップではなく、単年度のインセンティブ（業績賞与・手当）として設計
**設計思想**: 基準を高く設定し、頑張っている職員に大きく報いる「メリハリ型」

---

## 2. インセンティブ制度の設計

### 2.1 二層構造の概要

| 層 | 名称 | 対象 | 原資の性質 | 目的 |
|---|---|---|---|---|
| **第1層** | KPI連動インセンティブ | 特定部門・チーム | 増収に直結 → 自己回収型 | 収益向上の直接的動機付け |
| **第2層** | 人事評価連動インセンティブ | 全職員 | 粗利益からの配分 | 組織全体の底上げ・人材定着 |

### 2.2 第1層：KPI連動インセンティブ（部門別）

**特徴**: 達成基準を高めに設定し、経営貢献が明確な成果に報いる

| KPI項目 | 対象者 | 達成基準（高水準） | 支給額/回 | 頻度 | 年間最大額 |
|---|---|---|---|---|---|
| ダヴィンチ手術件数 | 手術チーム（外科・泌尿器・看護・ME） | 月15件以上（現状10.2件） | 50万円（チーム配分） | 月次 | 600万円 |
| 救急入院率 | 救急科・関連診療科・看護 | 入院率50%以上（現状45.7%） | 30万円（チーム配分） | 月次 | 360万円 |
| 紹介件数回復 | 地域連携室・整形外科 | 月間紹介件数が前年同月超 | 20万円（チーム配分） | 月次 | 240万円 |
| 算定漏れ防止 | 医事課・各病棟 | 施設基準充足率100%維持 | 30万円（チーム配分） | 四半期 | 120万円 |
| 病床稼働率 | 病棟看護・患者サポートセンター | 月間稼働率92%以上（現状88.9%） | 30万円（チーム配分） | 月次 | 360万円 |
| **第1層 年間合計** | | | | | **最大1,680万円** |

**第1層の自己回収メカニズム**:
- ダヴィンチ月15件 = 年180件。現状123件（月10.2×12）からの増加57件 × 手術増収30～50万/件 = **+1,710～2,850万円の増収** → 原資600万円を十分カバー
- 稼働率92%達成 = 1日平均294人（現状284.5人） → 入院収入+約1億円/年 → 原資360万円を大幅にカバー

### 2.3 第2層：人事評価連動インセンティブ（全職員対象）

**特徴**: 全職員が対象だが、評価基準を高く設定し上位者に厚く配分する「メリハリ型」

#### 評価ランクと分布・支給額

|---|---|---|---|---|---|
| **S（卓越）** | 部門目標を大幅超過し、組織全体に波及効果をもたらした | **5%** | 35名 | 月額5万円相当 | **60万円** |
| **A（優秀）** | 部門目標を超過達成し、改善提案を実行に移した | **15%** | 105名 | 月額2.5万円相当 | **30万円** |
| **B+（良好）** | 部門目標を達成し、安定した成果を出した | **25%** | 175名 | 月額1万円相当 | **12万円** |
| **B（標準）** | 通常業務を遂行 | 45% | 315名 | — | 0円 |
| **C（要改善）** | 目標未達 | 10% | 70名 | — | 0円 |

## 3. シナリオ別シミュレーション

### シナリオ1：保守的（増収なし・現状維持の場合）

経営改善施策の効果がまだ出ていない初年度を想定。

| 項目 | 金額 |
|---|---|
| 大橋病院の粗利益（現状） | 1.2億円 |
| 第1層 KPIインセンティブ | △1,000万円（部分達成を想定） |
| 第2層 人事評価インセンティブ | △5,000万円（圧縮版） |
| **インセンティブ総額** | **△6,000万円** |
| **インセンティブ後の粗利益** | **6,000万円** |
| **粗利益率** | **0.4%** |
| 人件費増加率 | +0.8% |

| 第2層の圧縮版 | 人数 | 年間支給額/人 | 小計 |
|---|---|---|---|
| S | 35名 | 40万円 | 1,400万円 |
| A | 105名 | 20万円 | 2,100万円 |
| B+ | 175名 | 8.5万円 | 1,500万円 |
| 合計 | 315名 | — | 5,000万円 |

**評価**: 粗利益の半分をインセンティブに充てる形。利益は残るが余裕は少ない。初年度のテスト導入としては妥当。Sランクでも年40万円と「大きく報いる」には物足りない。

---

### シナリオ2：標準（経営改善+1.5億円の増収を達成した場合）

外来患者数の一部回復、ダヴィンチ拡大、算定漏れ防止などが奏功した場合。

| 項目 | 金額 |
|---|---|
| 大橋病院の粗利益（改善後） | 2.7億円（現状1.2億+増収1.5億） |
| 第1層 KPIインセンティブ | △1,500万円（大部分達成） |
| 第2層 人事評価インセンティブ | △7,350万円（フルスペック） |
| **インセンティブ総額** | **△8,850万円** |
| **インセンティブ後の粗利益** | **1.8億円** |
| **粗利益率** | **1.1%** |
| 人件費増加率 | +1.2% |

**評価**: 増収の成果を職員に還元しつつ、粗利益1.8億円を確保。Sランクは年60万円（月5万円相当）、Aランクでも年30万円と、**頑張った職員に明確に報いる水準**。粗利益率も改善前（0.8%）を上回り、経営と人材投資の好循環を実現。

---

### シナリオ3：積極的（経営改善+3億円の増収を達成した場合）

全施策が順調に進捗し、診療報酬改定効果も加わった場合。

| 項目 | 金額 |
|---|---|
| 大橋病院の粗利益（大幅改善後） | 4.2億円（現状1.2億+増収3億） |
| 第1層 KPIインセンティブ | △1,680万円（ほぼ全達成） |
| 第2層 人事評価インセンティブ | △1億500万円（増額版） |
| **インセンティブ総額** | **△1億2,180万円** |
| **インセンティブ後の粗利益** | **3.0億円** |
| **粗利益率** | **1.9%** |
| 人件費増加率 | +1.7% |

| 第2層の増額版 | 人数 | 年間支給額/人 | 小計 |
|---|---|---|---|
| **S** | **35名** | **100万円** | **3,500万円** |
| **A** | **105名** | **40万円** | **4,200万円** |
| B+ | 175名 | 16万円 | 2,800万円 |
| 合計 | 315名 | — | 1億500万円 |

**評価**: **Sランク年100万円（月8.3万円相当）** は「大きく報いる」に値する水準。Aランク年40万円もモチベーションに十分寄与。インセンティブ総額1.2億円は人件費71億円の1.7%であり、同クラス平均（人件費率51.7%）との差10.5ptの中で十分吸収可能。粗利益3.0億円を確保しつつ、人材投資も実現する理想的なシナリオ。

## 5. 人事評価基準の設計（高水準設定）

### 5.1 職種別の評価項目と基準

**「頑張っている」を客観的に定義し、基準を高く設定する。**

#### 医師

| 評価項目 | B（標準） | A（優秀） | S（卓越） |
|---|---|---|---|
| 手術件数 | 前年同数 | 前年比+10%以上 | 前年比+20%以上 |
| 新入院患者数 | 予算達成 | 予算比+5%以上 | 予算比+10%以上 |
| 紹介件数 | 前年維持 | 前年比増加 | 前年比+15%以上 |
| 後進育成 | 研修医指導実施 | 学会発表指導+論文投稿 | 指導した研修医がアワード受賞等 |
| 地域連携 | 紹介元対応 | 訪問活動月2件以上 | セミナー企画・主催 |

#### 看護師

| 評価項目 | B（標準） | A（優秀） | S（卓越） |
|---|---|---|---|
| 重症度・看護必要度の記録精度 | 基準充足 | エラー率0.5%以下 | エラー率0%+他者指導 |
| 退院支援 | 計画通り実施 | 退院調整日数の短縮 | 在院日数短縮に病棟レベルで貢献 |
| インシデント・アクシデント | 報告遵守 | 予防的取組み提案 | 部署横断的な改善活動リード |
| 業務改善 | 通常業務遂行 | 改善提案1件以上/半期 | 提案が院内採用され効果実証 |
| 後輩育成 | プリセプター遂行 | 新人の離職率0% | 教育プログラム企画・実施 |

#### コメディカル・事務職

| 評価項目 | B（標準） | A（優秀） | S（卓越） |
|---|---|---|---|
| 業務目標達成 | 目標達成 | 目標超過（110%以上） | 目標超過+業務範囲の拡大 |
| 算定・請求精度（医事課） | エラー率基準内 | エラー率50%削減 | 算定漏れ発見+増収提案 |
| 効率化・DX推進 | 既存業務遂行 | 業務効率化提案1件以上 | DXツール導入・定着に主導的役割 |
| チーム貢献 | チーム内協力 | 部署横断的な活動参加 | 委員会・プロジェクトの主導 |

### 5.2 Sランクの基準イメージ

**Sランクは「自分の仕事を超えて、組織全体に価値をもたらした人」にのみ付与。**

具体例：
- 整形外科の医師が紹介元への訪問活動を月5件以上行い、紹介件数を前年比+20%回復させた
- 病棟看護師長が退院支援フローを再設計し、病棟の平均在院日数を0.5日短縮させた
- 医事課スタッフが施設基準管理システムの運用を主導し、算定漏れゼロを達成した
- 臨床工学技士がダヴィンチ新術式のトレーニングプログラムを設計し、スムーズな導入を実現した

## 7. リスクと対策

| リスク | 影響 | 対策 |
|---|---|---|
| 増収が計画を下回り原資不足 | インセンティブ額の縮小 → 職員の失望 | **粗利益連動型**にすることで、原資が自動調整される設計とする |
| 評価の公平性への不信 | モチベーション低下 | 評価基準の事前公開、評価者研修、異議申立て制度 |
| Sランクの乱発 | 原資超過、制度の形骸化 | Sランクは**院長承認制**（部門長推薦→院長決裁）とし、上限5%を厳守 |
| 部門間の不公平感 | 特定部門にインセンティブが偏る | 第1層（部門KPI）と第2層（全職員評価）の二層構造で全員にチャンスを保証 |
| 人件費増が法人の中期計画と矛盾 | 法人からの指摘 | インセンティブは変動費であり、増収に連動して支給 → 増収なければ支給なし、と説明 |

## 8. 推奨：シナリオ2（標準）での段階的導入

**理由:**
1. **Sランク年60万円、Aランク年30万円**は「頑張りに大きく報いる」水準として説得力がある
2. インセンティブ総額8,850万円は、+1.5億円の増収に対する**還元率59%**であり、職員と経営の適切なバランス
3. 粗利益1.8億円（率1.1%）を確保でき、法人への貢献も維持
4. 初年度はシナリオ1（保守）で導入し、増収が確認されたらシナリオ2に移行する**段階的アプローチ**が最も安全
5. 粗利益連動型とすることで、**収益が上がれば自動的にインセンティブも増える好循環**を構築
