# 学校法人東邦大学 経営環境分析レポート

**作成日**: 令和8年4月11日
**分析対象**: 法人全体の経営状況（令和7年度予算・中期経営計画・上期実績・組織目標）
**目的**: 大橋病院の経営改善に資する上位概念としての法人経営環境の把握

---

## 1. エグゼクティブサマリー

### 総合評価

学校法人東邦大学は、**創立100周年を迎える節目の年に、基本金組入前当年度収支差額△10.7億円の赤字予算**を組んでいる（令和7年度予算の概要）。赤字の主因は、大森地区再開発事業（新外来棟建設）を中心とする大型設備投資と、コロナ関連補助金の全廃・物価高騰による経費増嵩である。

新中期経営計画（R7-R9）では、**3年間の基本金組入前当年度収支差額はわずか0.48億円の黒字**にとどまり（法人全体収支計画）、R7の△10.7億円赤字をR8以降の黒字で取り戻す「V字回復型」の計画となっている。しかし、この計画の実現には**医療収入の着実な成長（3年間累計2,369億円、前回実績比+10.9%）**が不可欠であり、3病院の経営改善が法人全体の命運を握る構造である。

大橋病院は、3病院中で唯一粗利益が前年度比で改善（+122.4%）しており、法人を支える重要な柱である。ただし、設備投資の63%が大森再開発に集中する中、大橋病院は**限られた投資の中で自力での収益最大化**が求められる厳しいポジションにある。

## 2. 法人収支構造分析

### 2.1 収入構造 — 医療収入への高依存

【事実】令和7年度予算における法人の教育活動収入は999.9億円で、その**77.6%にあたる775.6億円が医療収入**である（事業活動収支予算書）。事業活動収入全体（教育活動外収入・特別収入を含む）は1,013.3億円。

|---|---|---|---|---|

【事実】医療収入の内訳は、入院収入491.1億円（63.3%）、外来収入281.8億円（36.3%）、雑医療収入2.7億円（0.4%）である（事業活動収支予算書）。

【解釈】法人収入の8割近くを医療収入に依存する構造は、3病院の経営パフォーマンスが法人全体の財務健全性を直接左右することを意味する。補助金は前年比+10.8億円と大幅増だが、これは生命理工学部構想関連の特殊要因（約20億円の補助金確定）を含む可能性が高く、持続的な収入源とは言い切れない。

### 3.2 V字回復計画の実現可能性

【事実】中期計画はR7の△10.7億円赤字から、R8に+3.5億円、R9に+7.7億円の黒字に転換する「V字回復」を描いている。

【解釈】V字回復の前提条件を検証する。

**収入面**: 医療収入はR7: 776億円→R8: 789億円→R9: 804億円と、毎年+13～15億円の増収を見込む。これは年率+1.7%～+1.9%の成長率であり、R7上期実績（予算比△0.4%）を踏まえると**楽観的な前提**と言わざるを得ない。

**支出面**: 人件費はR8に△5億円の減少を見込むが、医療経費は毎年+6～7億円の増加が計画されている。人件費削減は働き方改革・タスクシフトの推進が前提だが、医療の質を維持しつつの削減には限界がある。

**粗利益の趨勢**: 3年合計の粗利益177億円は前回実績比△15.4%（△32億円）であり、**収益力の低下傾向**が明確である。医療経費の増加（+12.6%）が医療収入の増加（+10.9%）を上回る構造が固定化している。

### 3.4 キャッシュフロー計画 — 流動性への注意

【事実】3年間のキャッシュフロー計画では、設備投資306.8億円 + 長期借入金返済18.2億円 = 325.0億円の資金需要に対し、減価償却費169.5億円 + 基本金組入前収支差額0.5億円 = 170.0億円の内部資金で賄い、**不足分155.0億円を手許現預金から取り崩す**計画である（法人全体収支計画）。

【解釈】3年間で手許資金を155億円取り崩す計画は、**法人の流動性を大幅に毀損**する。R6末時点の繰越支払資金が約243億円とすれば、R9末には約88億円にまで減少する計算である。これは突発的な設備更新や医療事故・災害等のリスクに対するバッファが著しく薄くなることを意味する。新たな借入を極力抑える方針は健全だが、計画通りの医療収入が達成できない場合、資金ショートのリスクが顕在化する。

#### 主要財務指標

| 指標 | 数値 | 判定 | 出典 |
|---|---|---|---|
| **流動比率** | **258.8%** | ◎良好（200%超） | 流動資産399.6億/流動負債154.4億 |
| **自己資本比率** | **68.4%** | ◎良好 | 純資産977.0億/総資産1,428.5億 |
| **負債比率** | **46.2%** | ○健全（100%以下） | 負債451.5億/純資産977.0億 |
| **固定長期適合率** | **80.8%** | ◎良好（100%以下） | 固定資産1,028.9億/(純資産977.0億+固定負債297.2億) |
| **借入金依存度** | **4.5%** | ◎低水準 | 借入金63.7億/総資産1,428.5億 |

【解釈】

**財務基盤は総じて健全である。** 自己資本比率68.4%、流動比率258.8%と、いずれも安全な水準にある。借入金は63.7億円（総資産比4.5%）と低水準で、前年度比12.2億円の着実な返済が進んでいる。

**ただし、以下の構造的特徴に留意が必要：**

1. **退職給与引当金238.1億円**: 負債の52.7%を占める最大の固定負債。今後の退職者増加により膨張リスクがある
2. **繰越収支差額△972.9億円**: 学校法人特有の基本金制度による構造的マイナスだが、R6で△40.1億円の悪化（基本金組入76.8億円が主因）。これは大森再開発等の設備投資が基本金に組み入れられる結果であり、実態的な財務悪化とは異なるが、悪化傾向の加速は注視すべき
3. **現金預金258.6億円 → R7末168.5億円（予算）→ R9末推定88億円**: 中期計画で155億円を取り崩す計画は、**現在の手許資金の6割を使い切る**規模。金融資産全体（342.4億円）を含めればバッファはあるが、有価証券・信託の売却には時間と手数料がかかる
4. **有形固定資産940.7億円のうち土地229.0億円**: 大森・習志野・佐倉の各キャンパス用地を含む。大森再開発で建物価値は今後増加するが、土地は横ばい
