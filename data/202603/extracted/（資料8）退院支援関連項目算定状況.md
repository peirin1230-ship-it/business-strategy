## ページ1

### 入退院支援加算

- **入退院支援加算1**: 700点
- **入院時支援加算1**: 230点（退院時1回）
- **地域連携診療計画加算**: 300点
- **総合機能評価加算**: 50点

#### 入退院支援加算1 算定要件

| 項目 | 内容 |
|---|---|
| 退院困難な患者の早期抽出 | 3日以内に退院困難な患者を抽出 |
| 入院早期の患者・家族との面談 | 7日以内に患者・家族と面談 |
| 多職種によるカンファレンスの実施 | 7日以内にカンファレンスを実施 |
| 退院調整部門の設置 | 専従1名（看護師又は社会福祉士） |
| 病棟への退院支援職員の配置 | 退院支援業務等に専従する職員を病棟に専任で配置（2病棟に1名以上） |
| 医療機関間の顔の見える連携の構築 | 連携する医療機関等（20か所以上）の職員と定期的な面会を実施（3回/年以上） |
| 介護保険サービスとの連携 | 介護支援専門員との連携実績 |

#### 入院時支援加算1 算定要件

入院前にアからクまでを全て実施して療養支援の計画書を作成した場合

| 記号 | 内容 |
|---|---|
| ア | 身体的・社会的・精神的背景を含めた患者情報の把握 |
| イ | 入院前に利用していた介護サービス又は福祉サービスの把握 |
| ウ | 褥瘡に関する危険因子の評価 |
| エ | 栄養状態の評価 |
| オ | 服薬中の薬剤の確認 |
| カ | 退院困難な要因の有無の評価 |
