# 病院経営分析リポジトリ

東邦大学医療センター大橋病院の医事統計データに基づく経営分析リポジトリです。
毎月の3病院合同病院運営戦略会議資料を蓄積し、多角的な経営分析を行います。

## 対象病院

| 病院 | 病床数 | 役割 |
|---|---|---|
| **大橋病院** | 320床 | 主分析対象 |
| 大森病院 | 800床 | グループ内ベンチマーク |
| 佐倉病院 | 380床 | グループ内ベンチマーク |

**大橋病院の特性**: DPC標準病院群 / 急性期一般入院料1 / 急性期充実体制加算 / 地域医療支援病院 / ICU 6床・HCU 12床・SCU 6床

## 大橋病院のポジション（2025年度2月時点）

### 同クラス病院との比較

同クラスの急性期一般入院料1算定病院（n=270、平均271床）と比較した大橋病院の立ち位置です。

| 指標 | 大橋病院 | 急性期1平均(n=270) | 対平均比 |
|---|---|---|---|
| 病床稼働率 | 88.9% | 77.5% | **+11.4pt** |
| 平均在院日数 | 8.9日 | 12.9日 | **69%** |
| 入院単価（1日） | 100,719円 | 71,439円 | **141%** |
| 外来単価（1日） | 22,628円 | 17,018円 | **133%** |
| 人件費率 | 41.2% | 51.7% | **△10pt** |
| 医業利益率 | 0.8% | △3.5% | **黒字維持** |

> 同クラスの56.7%が赤字の中、「短在院日数・高入院単価・高稼働率」で黒字を維持。
> 出典: WAM「病院の経営状況等について」(2023年度決算)、厚労省「令和6年病院報告」

### 前年度対比の実績（4-2月累計）

| 指標 | 前年度 | 今年度 | 前年比 |
|---|---|---|---|
| 収入合計 | 152.9億円 | 158.4億円 | **+3.6%** |
| 粗利益 | 5,530万円 | 1億2,299万円 | **+122.4%** |
| 入院単価 | 97,592円 | 101,517円 | **+4.0%** |
| 外来単価 | 21,551円 | 22,841円 | **+6.0%** |
| 手術件数（年度合計） | 6,162件 | 6,336件 | **+2.8%** |
| 人件費 | 64.97億円 | 65.19億円 | **+0.3%** |

> 予算対比では△3.2億円の未達だが、前年度対比では粗利益が2倍以上に改善。
> 3病院中、粗利益が前年度比で改善したのは大橋のみ（大森△32.9%、佐倉△26.4%）。

### 経営課題

- **外来患者数の減少**: 累計△8.5%（1日平均852人 vs 予算910人）。ただし単価+11.6%で総額は+2.1%
- **診療科ポートフォリオの二極化**: 化学療法3科（外科・泌尿器科・消化器内科）+3,924万円/月 vs 手術3科（整形外科・脳神経外科・心臓血管外科）△5,672万円/月
- **整形外科の紹介激減**: △543件（全科最大）→ 外来+入院合算で△2,492万円/月の減収
- **令和8年度改定の増収見込み**: DPC係数改定等で約+3億円/年

## 成果物一覧

### 定例成果物（毎月作成）

| 成果物 | 内容 | 主な読者 |
|---|---|---|
| **経営分析レポート** | 予算対比＋前年度対比の定量分析、3病院比較、診療科別分析 | 事務部長、経営企画 |
| **幹部説明原稿** | 資料番号順の口頭説明スクリプト。簡潔・事実中心 | 会議での説明者 |
| **想定Q&A集** | 幹部からの質問を想定し、データ根拠に基づく回答を準備 | 会議での説明者 |
| **インサイトレポート** | 5視点統合分析（データ・臨床・負荷・請求・経営） | 診療科長、看護部長、医事課 |
| **議事メモ** | 論点ベースの会議記録。仮説の進化とネクスト論点を含む | 会議参加者、経営幹部 |

### インサイトレポートの5視点フレームワーク

| 視点 | 内容 | 主な読者 |
|---|---|---|
| データファクト | 数値事実と出典 | 全員 |
| 臨床現場の視点 | 医師・看護師・コメディカルの業務への影響 | 診療科長、看護部長 |
| 現場負荷の評価 | 提言実行時の業務負担増減と緩和策 | 看護部長、病棟師長 |
| 保険請求の視点 | 算定コード・算定要件・査定リスク・算定漏れ | 医事課スタッフ |
| 経営アクション | 収益インパクトの定量化と実行計画 | 事務部長、経営企画 |

### 議事メモのフォーマット（高松智史式「議事録進化論」ステージ7）

テーマベースの「議事録」ではなく、論点ベースの「**議事メモ**」を作成します。

| 要素 | 内容 |
|---|---|
| 大論点（BIG QUESTION） | この会議で答えるべき問い |
| 論点別ディスカッション | 各論点の仮説の進化（Before→After）、空気感、ステータス |
| 未議論の論点 | 本来議論すべきだったが触れられなかった論点 |
| ネクスト論点 | 次に解くべき「問い」（TASKではなく論点として記述） |
| ネクストステップ | 具体的なTASK（担当・期限・紐づく論点） |
| 空気感メモ | 会議全体のトーン・温度感 |

## PDF生成

`generate_pdf.py` でMarkdownからPDFを一括生成できます。

```bash
python generate_pdf.py
```

- フォント: メイリオ（`C:/Windows/Fonts/meiryo.ttc`）
- 依存: `fpdf2`（`pip install fpdf2`）

## 月次運用手順

1. `data/YYYYMM/raw/` に戦略会議資料（PDF）を格納
2. PDFを抽出し `data/YYYYMM/extracted/` に構造化データを保存
3. 分析を実施し `analysis/YYYYMM/` 配下の各フォルダに成果物を格納
4. 同クラス病院との外部ベンチマーク比較を実施
5. データ整合性チェックを実施（全数値を原本資料と突合）
6. `python generate_pdf.py` で最終レポートをPDF出力
7. 会議後、議事メモを `analysis/YYYYMM/meeting/` に格納（高松式フォーマット）

## 分析ルール

詳細は `.claude/CLAUDE.md` を参照。主なポイント：

- **データと根拠の厳格な分離**: 事実【事実】と解釈【解釈】を明確に区分
- **予算対比＋前年度対比の併記**: 予算未達でも前年度改善なら明記する
- **期間の一致**: 前年度対比は必ず同一期間で比較（11ヶ月 vs 12ヶ月の混同禁止）
- **3病院比較の深掘り**: 大橋病院の劣位指標を重点分析
- **診療科別ポートフォリオ分析**: 成長科と縮小科を分類し、費目別に要因を推察
- **同クラス病院との外部ベンチマーク**: 急性期1算定病院(n=270)等の全国データと比較
- **現場負荷への配慮**: 全提言に看護師・コメディカルの業務負担評価を付記
- **品質保証**: 成果物完成後に全数値を原本資料と突合してから最終版とする
