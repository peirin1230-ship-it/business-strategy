# 病院経営分析リポジトリ

東邦大学医療センター大橋病院の医事統計データに基づく経営分析リポジトリです。
毎月の3病院合同病院運営戦略会議資料を蓄積し、多角的な経営分析を行います。

## 対象病院

- **大橋病院**（主分析対象・320床）
- 大森病院（ベンチマーク比較）
- 佐倉病院（ベンチマーク比較）

## ディレクトリ構成

```
business-strategy/
├── data/                          # 月次データ
│   └── YYYYMM/
│       ├── raw/                   # 原本PDF（戦略会議資料）
│       └── extracted/             # PDF抽出データ（.md, .csv）
│
├── analysis/                      # 経営分析成果物
│   └── YYYYMM/
│       ├── report/                # 経営分析レポート（PDF出力）
│       ├── presentation/          # 幹部説明原稿
│       ├── qa/                    # 想定Q&A集
│       └── insights/              # インサイトレポート
│
├── generate_pdf.py                # Markdown→PDF変換スクリプト
└── README.md
```

## データ層（data/）

### raw/ — 原本PDF

毎月の戦略会議資料（PDF）をそのまま格納します。

| 資料番号 | 内容 | 主な分析テーマ |
|---|---|---|
| 資料1 | 3病院収支比較表 | 収支構造、粗利益率、人件費比率 |
| 資料1-1 | 医事統計 | 入院・外来患者数、単価、稼働率 |
| 資料1-2 | 診療科別前年比較 | 診療科別の収益動向 |
| 資料2 | 外来化学療法加算算定状況 | 化学療法の成長分析、曜日別稼働 |
| 資料3 | 手術資料 | 手術件数、全麻率、緊急手術 |
| 資料4 | 特定入院料算定状況 | ICU/HCU/SCU稼働率・算定率 |
| 資料5 | リハビリテーション算定状況 | リハビリ種別の単位数・金額推移 |
| 資料6 | 後発医薬品指数 | ジェネリック使用率、金額ベース |
| 資料7 | 地域支援病院 紹介率推移 | 紹介・逆紹介件数、診療科別 |
| 資料7-1/7-2 | 病診連携室資料 | 紹介元別増減、受入拒否分析 |
| 資料8 | 退院支援関連項目算定状況 | 入退院支援加算、病棟別算定 |
| 資料9 | 放射線治療算定状況 | がん種別治療件数、IMRT推移 |
| 資料10 | 3病院指摘検討事項 | KPI進捗（査定率、謝絶率等） |

### extracted/ — 抽出データ

PDFから抽出した構造化データ（Markdown, CSV）を格納します。
分析の再現性と効率化のために、原本とは分離して管理します。

## 分析層（analysis/）

### report/ — 経営分析レポート

データに基づく定量分析レポート。最終版はPDF形式で出力。

- 3病院比較（大森・大橋・佐倉のベンチマーク分析）
- 収支・入院・外来・各部門の月次分析

### presentation/ — 幹部説明原稿

口頭説明用スクリプト。資料番号順に沿って解説し、簡潔な言葉遣いで事実を淡々と述べるスタイル。

### qa/ — 想定Q&A集

幹部から想定される質問とデータ根拠に基づく回答。

### insights/ — インサイトレポート

データ分析から分離した示唆・仮説・提言。**5視点統合分析フレームワーク**で構成。

| 視点 | 内容 | 主な読者 |
|---|---|---|
| データファクト | 数値事実と出典 | 全員 |
| 臨床現場の視点 | 医師・看護師・コメディカルの業務への影響 | 診療科長、看護部長 |
| 現場負荷の評価 | 提言実行時の業務負担増減と緩和策 | 看護部長、病棟師長 |
| 保険請求の視点 | 算定コード・算定要件・査定リスク・算定漏れ | 医事課スタッフ |
| 経営アクション | 収益インパクトの定量化と実行計画 | 事務部長、経営企画 |

## PDF生成

`generate_pdf.py` でMarkdownからPDFを一括生成できます。

```bash
python generate_pdf.py
```

- フォント: メイリオ（`C:/Windows/Fonts/meiryo.ttc`）
- 依存: `fpdf2`（`pip install fpdf2`）

## 月次運用手順

1. `data/YYYYMM/raw/` に戦略会議資料（PDF）を格納
2. PDFを抽出し `data/YYYYMM/extracted/` に構造化データを保存
3. 分析を実施し `analysis/YYYYMM/` 配下の各フォルダに成果物を格納
4. 品質検証レビューを実施（データ正確性・論理一貫性・実務有用性）
5. `python generate_pdf.py` で最終レポートをPDF出力

## 分析ルール

詳細は `.claude/CLAUDE.md` を参照。主なポイント：

- **データと根拠の厳格な分離**: 事実と解釈・仮説・提言を明確に区分
- **3病院比較の深掘り**: 大橋病院の劣位指標を重点分析
- **現場負荷への配慮**: 全提言に看護師・コメディカルの業務負担評価を付記
- **品質保証**: 成果物完成後に批判的レビューを実施してから最終版とする
