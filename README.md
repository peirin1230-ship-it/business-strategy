# 経営戦略リポジトリ

医事統計データに基づく病院経営分析のためのリポジトリです。
毎月の戦略会議資料を蓄積し、経営分析を行います。

## ディレクトリ構成

```
business-strategy/
├── data/              # 月次戦略会議資料（YYYYMM形式）
│   └── 202603/        # 2026年3月分
├── analysis/          # 経営分析レポート（YYYYMM形式）
│   └── 202603/        # 2026年3月分析結果
└── README.md
```

## 運用ルール

### データ格納（data/YYYYMM/）
毎月の戦略会議資料（PDF）を `data/YYYYMM/` ディレクトリに格納します。

含まれる資料：
- 医事統計
- 診療科別前年比較
- 3病院収支比較表
- 外来化学療法加算算定状況
- 手術資料
- 特定入院料算定状況
- リハビリテーション算定状況
- 後発医薬品指数
- 地域支援病院（紹介率）推移・病診連携室資料
- 退院支援関連項目算定状況
- 放射線治療算定状況
- 3病院指摘検討事項

### 経営分析（analysis/YYYYMM/）
月次データに基づく経営分析レポートを `analysis/YYYYMM/` に格納します。

## 月次データ追加手順

1. `data/YYYYMM/` ディレクトリを作成
2. 戦略会議資料（PDF）を格納
3. 分析実施後、`analysis/YYYYMM/` に分析レポートを格納
