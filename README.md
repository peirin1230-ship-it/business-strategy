# 経営戦略リポジトリ

医事統計データに基づく病院経営分析のためのリポジトリです。
毎月の戦略会議資料を蓄積し、経営分析を行います。

## ディレクトリ構成

```
business-strategy/
├── data/                          # 月次データ
│   └── YYYYMM/
│       ├── raw/                   # 原本PDF（戦略会議資料）
│       └── extracted/             # PDF抽出データ（.md, .csv）
│
├── analysis/                      # 経営分析成果物
│   └── YYYYMM/
│       ├── report/                # 経営分析レポート（PDF出力）
│       ├── presentation/          # 幹部説明原稿
│       ├── qa/                    # 想定Q&A集
│       └── insights/             # インサイト（示唆・仮説・提言）
│
├── .gitignore
└── README.md
```

## データ層（data/）

### raw/ - 原本PDF
毎月の戦略会議資料（PDF）をそのまま格納します。

含まれる資料：
- 資料1: 3病院収支比較表
- 資料1-1: 医事統計
- 資料1-2: 診療科別前年（月平均）比較
- 資料2: 外来化学療法加算算定状況
- 資料3: 手術資料
- 資料4: 特定入院料算定状況
- 資料5: リハビリテーション算定状況
- 資料6: 後発医薬品指数
- 資料7: 地域支援病院（紹介率）推移・病診連携室資料
- 資料8: 退院支援関連項目算定状況
- 資料9: 放射線治療算定状況
- 資料10: 3病院指摘検討事項

### extracted/ - 抽出データ
PDFから抽出した構造化データ（Markdown, CSV）を格納します。
分析の再現性と効率化のために、原本とは分離して管理します。

## 分析層（analysis/）

### report/ - 経営分析レポート
データに基づく定量分析レポート。最終版はPDF形式で出力。
- 3病院比較（特に劣位指標の深掘り）
- 収支・入院・外来・各部門の分析

### presentation/ - 幹部説明原稿
口頭説明用スクリプト。話す順序・強調ポイント・時間配分を明記。

### qa/ - 想定Q&A集
幹部から想定される質問とデータ根拠に基づく回答。

### insights/ - インサイト
データ分析から分離した示唆・仮説・提言。
根拠データと推論の飛躍部分を明示。

## 月次運用手順

1. `data/YYYYMM/raw/` に戦略会議資料（PDF）を格納
2. PDFを抽出し `data/YYYYMM/extracted/` に構造化データを保存
3. 分析を実施し `analysis/YYYYMM/` 配下の各フォルダに成果物を格納
4. 批判的レビューを実施し、品質を担保してから最終版とする
5. 最終レポートをPDF形式で出力
